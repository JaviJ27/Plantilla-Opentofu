# Plantilla-Opentofu
Una plantilla para hacer maquinas virtuales o escenarios con opentofu

Esta plantilla se debe implementar despues de clonar https://github.com/JaviJ27/opentofu-libvirt-josedomingo

Tras clonar dicho repositorio, esta plantilla se debe clonar dentro de ~/opentofu-libvirt-josedomingo/MaquinasPersonalizadas/ (Hay que crear la carpeta MaquinasPersonalizadas)
